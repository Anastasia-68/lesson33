# Gulp SCSS Project

Учебный проект с кастомной конфигурацией Gulp.

## Реализованные возможности
- Работа с препроцессором SCSS
- Автоматическое обновление страницы (BrowserSync)
- Добавление вендорных префиксов (Autoprefixer)
- Генерация базового и минифицированного CSS
- Минимизация стилей

## Структура проекта
- src/ — исходные файлы (HTML, SCSS)
- dist/ — скомпилированные CSS-файлы
- gulpfile.js — конфигурация Gulp

## Запуск проекта
```bash
npm install
npx gulp
